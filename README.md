# Kopi Klasik

Kopi Klasik adalah aplikasi web berbasis PWA (Progressive Web App) untuk e-commerce makanan & minuman. Aplikasi ini dibangun menggunakan React, TypeScript, dan Tailwind CSS dengan fokus pada performa, UI modern, dan pengalaman pengguna yang optimal.

---

## Tech Stack

* React 18
* TypeScript
* Vite
* Tailwind CSS
* shadcn/ui + Radix UI
* TanStack React Query
* Zustand
* Framer Motion
* Supabase
* Vite PWA Plugin

---

## Fitur Utama

### Authentication

* Login & Register
* Integrasi Supabase Auth
* Manajemen state menggunakan Zustand

### E-Commerce

* List produk
* Detail produk
* Keranjang (Cart)
* Checkout

### Cart System

* Tambah / hapus produk
* Update quantity
* State global

### Favorites

* Menyimpan produk favorit

### Address Management

* Menyimpan alamat user
* Digunakan saat checkout

### PWA Features

* Install ke device
* Offline support (basic)
* Mobile-friendly

### UI/UX

* Responsive design
* Komponen modern (Radix UI)
* Animasi dengan Framer Motion
* Toast notification

---

## Instalasi & Menjalankan Project

1. Clone repository
   cd kopi-klasik

3. Install dependencies
   npm install
   atau
   bun install

4. Jalankan project
   npm run dev
